Frequently Asked Questions

What is GoodreadsFilter?

GoodreadsFilter allows you to filter the books associated with your Goodreads account by a specified filter (Author) and filter constraint (George R.R. Martin). As you can see, each filter is associated with a filter constraint and you are allowed to select multiple pairs when you choose to filter your Goodreads list.

Why can't I just enter the link of my Goodreads account and have my list of books filtered like that? Why do I have to use a CSV file?

This would require me to use either the Goodreads API or a parser (to parse the Goodreads website). Unfortunately, I cannot use the Goodreads API because Goodreads does not hand out API keys anymore. If I could use the Goodreads API, then I would. As for the parser, this is something that I have considered using and I may use it in the future. But for now, it is easier for me to filter the books associated with a Goodreads account using a CSV file.
